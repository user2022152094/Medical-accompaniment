一个基于 Vue3 的医疗陪诊服务平台，包含后台管理系统（PC端）与用户下单系统（移动端），实现陪护师管理、医院下单、订单流转等完整业务流程。

## ✨ 功能模块

### PC管理端
- 用户管理
- 菜单权限管理
- 陪护师管理
- 医院管理
- 订单管理

- ## 项目运行方式 ⭐
### 1. 克隆项目
git clone https://github.com/user2022152094/Medical-accompaniment.git

### 2. 启动PC端
- cd PC
- npm install
- npm run dev

### 3. 启动移动端
- cd pzH5
- npm install
- npm run dev
